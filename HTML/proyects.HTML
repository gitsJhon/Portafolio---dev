MyTour13

Esta es una aplicacion web destinada a mejorar el turismo en la comuna 13 de Medellin, en la cual los turistas que llegan a la comuna pueden organizar sus recorridos por esta antes de llegar, pueden encotrar un guia local y cotizar con este los precios que tienen cada uno

Para este proyecto utilice PHP nativo usando el modelo MVC (modelo vista - controlador) para el backedn, SQL para la base de datos del proyecto y en el frontend se utilizo HTML, CSS y JS utilizando una mezcla de mezcla de nativo y Boostrap dependiendo de la necesidad especifica del elemento a desarrolladr
